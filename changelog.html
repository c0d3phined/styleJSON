Version Version 1.2 Release Nov 3, 2011
Date parsing styleJSON will now accept and parse a formatted Date string (ex. "/Date(1314151709000)/") into a date object. By setting the defaults you can specify the path to the popular dateFormat script, as well as the desired formatting you want to format the date to.
You can set the defaults explicitly before executing styleJSON like below:
$.styleJSON.defaults.dateFormat = "dddd, mmmm d, yyyy";
$.styleJSON.defaults.dateFormatPath = 'js/date.format.js';
$.styleJSON.defaults.useDateFormat = true;
If you had not included the Date Format script in the page via <script ... /> tags, styleJSON will perform a GET (jQuery.getScript) to load the script at run-time.
